Block & Inline Elements

Block Elemets :-

"Block elements are the elements that takes the whole width of the webpage and start always from new line"

Examples-

  1. Address
  2. blockquote
  3. canvas

Inline Elements :-

"Inline elements are the elements that doesn't take the whole width of the webpage and it starts from the same line of the previous element"

Examples-

  1. mark
  2. q
  3. em
  • abc
    1. 1234
    2. 4567
    3. 6543
    4. 7890
